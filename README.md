# homebrew-toolkit

Homebrew tap for the [toolkit](https://github.com/henrymodisett/toolkit) CLI.

## Install

```bash
brew tap henrymodisett/toolkit
brew install toolkit
```

## Usage

```bash
toolkit new my-project      # Bootstrap a new project
toolkit update              # Update current project to latest toolkit
toolkit sync --pull-first   # Update all registered projects
toolkit doctor              # Health check
toolkit version             # Show version
```

## Update

```bash
brew upgrade toolkit
```
